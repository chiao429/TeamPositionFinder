# 小隊位置搜尋系統 - UI 設計規格

## 1. 設計概念

### 1.1 設計理念
- **簡潔明瞭**: 一目了然的介面，減少認知負擔
- **視覺引導**: 清晰的視覺層次，引導使用者操作流程
- **即時反饋**: 互動時提供即時的視覺回饋
- **現代美學**: 採用現代扁平化設計風格

### 1.2 設計原則
- 以使用者為中心
- 保持一致性
- 提供清晰的視覺回饋
- 優化載入效能

## 4. 間距系統

### 4.1 間距單位
基於 4px 網格系統

```
2  = 0.5rem = 8px
3  = 0.75rem = 12px
4  = 1rem = 16px
6  = 1.5rem = 24px
8  = 2rem = 32px
12 = 3rem = 48px
```

### 4.2 元件間距
- 主要區塊間距: 32px (8)
- 卡片內部間距: 24px (6)
- 表單元素間距: 16px (4)
- 文字行距: 1.5-1.75

## 5. 元件設計

### 5.1 搜尋欄 (SearchBar)

#### 視覺設計
```
┌────────────────────────────────────────────────┐
│  [🔍] 請輸入小隊名稱...          [ 搜尋 ]    │
└────────────────────────────────────────────────┘
```

#### 規格
- **尺寸**: 寬度 100% (最大 672px), 高度 56px
- **邊框**: 2px solid gray-300
- **圓角**: 12px
- **陰影**: sm (輕微陰影)
- **焦點狀態**: 
  - 邊框變為 primary-500
  - 外圈 4px primary-100 光暈

#### 互動狀態
| 狀態 | 樣式 |
|------|------|
| 預設 | 灰色邊框 |
| 懸停 | 無變化 |
| 焦點 | 藍色邊框 + 光暈 |
| 輸入中 | 保持焦點樣式 |

### 6.1 斷點定義

| 斷點 | 寬度 | 裝置類型 |
|------|------|----------|
| sm | 640px | 手機 (橫向) |
| md | 768px | 平板 |
| lg | 1024px | 桌面 |
| xl | 1280px | 大螢幕 |

### 6.2 響應式調整

#### 手機 (< 640px)
- 座位方塊: 60px × 60px
- 字型縮小 10-20%
- 單欄佈局
- 減少間距

#### 平板 (640px - 1024px)
- 座位方塊: 70px × 70px
- 保持原始字型大小
- 2 欄資訊佈局

#### 桌面 (> 1024px)
- 座位方塊: 80px × 80px
- 完整佈局
- 最大寬度限制 (1280px)

## 7. 互動設計

### 7.1 搜尋流程

```
1. 使用者輸入小隊名稱
   ↓
2. 點擊搜尋按鈕 或 按下 Enter
   ↓
3. 系統搜尋資料 (< 500ms)
   ↓
4a. 找到結果:
    - 顯示小隊資訊卡片 (淡入動畫)
    - 座位高亮 (縮放 + 脈動動畫)
    
4b. 未找到:
    - 顯示錯誤提示 (淡入動畫)
    - 清除之前的高亮
```

### 7.2 動畫時序

| 動作 | 持續時間 | 緩動函數 |
|------|----------|----------|
| 淡入 | 300ms | ease-in-out |
| 縮放 | 200ms | ease-out |
| 脈動 | 2000ms | cubic-bezier |
| 顏色過渡 | 200ms | ease |

### 7.3 懸停效果

| 元素 | 效果 |
|------|------|
| 搜尋按鈕 | 背景加深 + 陰影增強 |
| 一般座位 | 背景變淺 |
| 高亮座位 | 無額外效果 |

## 8. 無障礙設計

### 8.1 鍵盤導航
- Tab: 在可互動元素間切換
- Enter: 觸發搜尋
- Esc: 清除搜尋結果

### 8.2 ARIA 標籤
```html
<input 
  aria-label="搜尋小隊名稱"
  aria-describedby="search-hint"
/>

<button 
  aria-label="執行搜尋"
  type="submit"
/>

<div 
  role="alert" 
  aria-live="polite"
>
  查無此小隊
</div>
```

### 8.3 對比度
- 文字與背景對比度 ≥ 4.5:1 (WCAG AA)
- 大文字對比度 ≥ 3:1
